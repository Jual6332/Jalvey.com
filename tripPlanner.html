BACK-END APPLICATION

TRIP-PLANNER

ROLE: LEAD DEVELOPER (C++)

ABOUT THE PROJECT

This back-end project was set up to simulate a trip-planning service like TripAdvisor. The project uses the Graph data structure to implement "Trip-Planning" functionality for tourists or vacationers. Using Dijkstra's algorithm and other shortest-distance algorithms, travel routes between cities are simplified. By implementing service locations for each city, travel routes are also personalized, allowing users to stop at their favorite restaurants, hotels, and other vacation spots between destinations.

"PLAN A TRIP" FEATURE


This feature calculates the shortest distance between the user's starting location and destination. Then application then suggests places to stop for gas, food, and other amenities.
