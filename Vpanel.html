Vortex Panel Method

Role: SOLE DEVELOPER (MATLAB)

ABOUT THE PROJECT

To calculate the potential flow for any airfoil, an Airfoil Generator was created to generate the geometry for any airfoil. By inputting the airfoil characteristics according to 4-digit NACA airfoil standards (NACA - max camber, max camber location, and chord thickness), the airfoil shape was generated using the camber equation for standard airfoils. This geometry was analyzed using the Vortex Panel Method implemented via the Kuethe-Chow algorithm. From this data, accurate calculations for pressure, lift, and drag on the airfoil could be made. These results were compared against thin airfoil-theory to determine computational validity.
